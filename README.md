# WPF Hotel Management System – C# Desktop Application

This project is a desktop application developed in C# using WPF as part of an academic course in Windows Systems Programming.

The system is designed for managing a cabin/hotel booking platform, implementing a layered architecture and multiple design patterns to ensure modularity, scalability, and maintainability.

## 🏗 System Architecture

The project follows a layered architecture:

- **BE (Business Entities)** – Core domain objects representing system entities
- **BL (Business Logic)** – Business rules and system operations
- **DAL (Data Access Layer)** – Data persistence and retrieval logic
- **DS (Data Source)** – In-memory or structured data storage
- **PL / UI (Presentation Layer)** – WPF user interface layer

This separation ensures clear responsibility distribution and maintainable code structure.

## 🧩 Design Patterns Used

The system incorporates several object-oriented design patterns:

- **Factory Pattern** – Used for object creation abstraction
- **Adapter Pattern** – Enables compatibility between different interfaces
- **Clone Pattern (Deep Copying)** – Supports safe duplication of objects
- **Enum Usage** – For structured representation of system states and categories
- **XML Serialization** – For data persistence and structured storage

## 💡 Key Features

- Management of cabin/hotel reservations
- Customer data handling

- Booking creation and management
- Structured data persistence using XML
- Modular and extensible system design
